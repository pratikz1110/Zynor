# Zynor Web (Next.js 14)

## Prerequisites
- Node.js 18+
- npm

## Install
```bash
cd apps/web
npm install
```

## Run (Dev)
```bash
npm run dev
```

Open http://localhost:3000

## Scripts
- dev: start Next.js in dev
- build: build production bundle
- start: start production server
- lint: run ESLint
- format: format with Prettier
- typecheck: run TypeScript checks

## Env
- NEXT_PUBLIC_API_URL (defaults to http://localhost:8000)

## Notes
- TailwindCSS and basic shadcn-style UI primitives included
- React Query is configured at the root layout
